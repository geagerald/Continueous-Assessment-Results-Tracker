# Continueous-Assessment-Results-Tracker
===========================================
a program in C++ that tracker a registered user's continueous assessment results
-----------------
A WELL DESIGNED PROGRAM USED TO HELP STUDENTS ALWAYS TO BE AWARE OF WHAT THEY ARE GETTING ON THERE CONTINUEOUS ASSESSMENT RESULTS THROUGHOUT THE YEAR. THIS IS TO HELP THE CALCULATE HOW THEY ARE PERFORMING AND CALCULATE THE WORK THEY NEDD TO PUT INTO NEED ORDERTO PASS

===========================================================================================================================================
A student gets to create un uccount in order to enter, manipulate, edit and delete continueous assessment results of a course thoughout the semister
-------------------------------------------------------------------------------------------------------------------------------------------
Logical Operations
--------------------
* able to automatically calculate grade as student enters a contineous assessment result
* deducts marks a student has remaining on his CA when the user a new assessment on the course
* auto validates the users assessment ebtry to make sure it does exid the total expected mark

FUNCTIONS
----------------
* allow user to enter a course and its assessment results
* allow user to delete un entire course
* allow user to delete a specific assessment result on a desired course ( while updating its total marks gained and remaining at the same time )
* allow user to view all course, course results, marks remaining on CA and grades gained on each at once
* usess files to store and retrieve data of the user
